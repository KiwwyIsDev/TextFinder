# TextFinder
**หากคุณต้องการค้นหาคำบางคำจากข้อมูลที่มีอยู่มากสามารถใช้ TextFinder ในการหาได้ เช่น**
*ข้อมูลที่ต้องการหา*

	username1
	username2
	username3
		

*ข้อมูลที่มี*
	
	username1:password1
	username3:password3
	username2:password2

*Output:*

	username1:password1
	username2:password2
	username3:password3
